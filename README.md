# todo-taskmaster
A high-performance, responsive Todo Application built with Vanilla JS, featuring a Glassmorphic UI, real-time search, and dynamic progress tracking.
TaskMaster Ultra is a lightweight, high-performance Todo application that bridges the gap between minimalist design and powerful functionality. Built entirely with Vanilla JavaScript, it handles 200+ concurrent tasks with smooth animations and zero framework overhead.

✨ Key Features
API Integration: Fetches and synchronizes 200 live todos from JSONPlaceholder.

Glassmorphism UI: A premium "frosted glass" aesthetic with responsive grid layouts.

Live Progress Analytics: An SVG-based radial dashboard that tracks your completion rate in real-time.

Advanced Filtering: Combined logic for Search, Incomplete, and Completed states.

Zero-Lag Performance: Staggered animation delays for a smooth, professional feel even with large datasets.

Smart Feedback: Integrated toast notification system for local task additions.

🛠️ Tech Stack
HTML5: Semantic structure.

CSS3: Custom properties (variables), Flexbox, Grid, and Backdrop-filters.

JavaScript: Modern ES6+ (Async/Await, Document Fragments, State Management).
